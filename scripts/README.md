# Scripts and Documentation

This folder contains all SQL scripts and related documentation for the healthcare provider directory application.

## Structure

### `/sql` - Database Scripts
- `community-features-schema.sql` - Database schema for community features (reviews, ratings, availability)
- `database-schema.sql` - Main database schema
- `favorites-schema.sql` - Schema for user favorites functionality
- `setup-community-features.sql` - Setup script for community features

### `/docs` - Documentation
- `CALL_RESULTS_INTEGRATION.md` - Documentation for call results integration
- `COMMUNITY_FEATURES_SETUP.md` - Setup guide for community features
- `LANDING_PAGE_IMPROVEMENTS.md` - Landing page improvement documentation
- `OMNIDIM_INTEGRATION_SUMMARY.md` - OmniDim integration summary
- `README-call-id-polling.md` - Call ID polling documentation
- `README-COMMUNITY-SETUP.md` - Community setup instructions
- `SUPABASE_FAVORITES_SETUP.md` - Supabase favorites setup guide

## Usage

To execute SQL scripts, use your preferred database client or command-line tool. Make sure to execute them in the correct order based on dependencies outlined in the documentation files.

For setup instructions, refer to the relevant documentation files in the `/docs` folder.
